# Assistente de Carreira para Dados (NotebookLM)

Projeto que utiliza IA para auxiliar candidatos iniciantes a conseguirem emprego na área de dados.

## Funcionalidades

- Análise de vagas
- Adaptação de currículo para ATS (Gupy)
- Simulação de entrevistas
- Sugestão de melhorias profissionais

## Tecnologias

- NotebookLM (IA)
- Estratégias de recrutamento

## Objetivo

Ajudar candidatos iniciantes a se posicionarem melhor no mercado de dados, sem parecerem presunçosos.

## Como usar

1. Inserir vaga desejada
2. Solicitar análise
3. Adaptar currículo
4. Treinar entrevistas
